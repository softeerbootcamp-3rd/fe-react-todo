### 기능 요구사항

기능 우선순위: 상 / 중 / 하

- (상) 투두리스트 등록
  - 할 일 내용 입력 가능
  - 투두리스트 내용 입력 시 버튼 활성화
  - “등록 버튼” 을 통해 투두리스트 등록
- (상) 투두리스트 삭제
  - 각 할 일의 “삭제 버튼”을 통해서 투드리스트 삭제
- (상) Json-server 연동
- (중) 투두리스트 완료 (체크박스)
  - 각 할 일의 체크박스 선택 시 회색 배경으로 변경
  - 이 때, 삭제 기능 가능
- (하)(선택) 날짜별 투두리스트 분리
